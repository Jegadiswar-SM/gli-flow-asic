# Development Workflow

## 1. Create Branch

```bash
git checkout -b feature/<feature-name>
```

---

## 2. Validate Environment

Run:

```bash
python3 environment/validation/run_full_validation.py
```

---

## 3. Validate Release Integrity

Run:

```bash
python3 release/validate_release.py
```

---

## 4. Commit Changes

Example:

```bash
git commit -m "Add execution observability improvement"
```

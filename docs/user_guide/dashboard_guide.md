# Dashboard Guide

The GLI-FLOW dashboard provides a visual interface for managing designs, monitoring runs, and analyzing results.

- **Runs Page**: List runs and view execution status.
- **Run Detail**: View detailed metrics and artifacts for a single run.
- **Failure Atlas**: Analyze and manage design failures.
- **Telemetry**: Monitor system health and telemetry events.
- **Settings**: Configure dashboard and API connection.

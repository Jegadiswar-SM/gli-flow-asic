# CLI Reference

This document lists the available GLI-FLOW commands.

## Production Commands
- `gli-flow run`: Run a design through the full RTL-to-GDS pipeline
- `gli-flow history`: Show execution history
- `gli-flow status`: Show recent run status
- `gli-flow batch`: Run multiple designs in parallel
- `gli-flow init`: Create a new design manifest
- `gli-flow quickstart`: Interactive setup wizard for new designs
- `gli-flow report`: Show QoR report for a completed ORFS run
- `gli-flow install`: Install gli-flow and toolchain dependencies
- `gli-flow ci`: Run a design in CI mode
- `gli-flow doctor`: Validate installed EDA toolchain
- `gli-flow reset-runs`: Delete all run history
- `gli-flow db`: Database schema management
- `gli-flow diagnose`: Diagnose a failed run
- `gli-flow show-telemetry`: Show exact telemetry payload
- `gli-flow config`: View/change GLI-FLOW configuration
- `gli-flow setup`: Interactive first-time setup
- `gli-flow support-bundle`: Generate support bundle
- `gli-flow predict`: Predict execution risk

## Experimental Commands
- `gli-flow remote`: Run design on remote machine
- `gli-flow cloud`: Cloud ingestion management
- `gli-flow investigate`: LLM investigation
- `gli-flow investigate-migrate`: Migrate investigations
- `gli-flow dashboard`: Start the dashboard
- `gli-flow upgrade-check`: Check for newer versions
- `gli-flow ai-assist`: AI Investigation Assistant
- `gli-flow escalate`: Community escalation
- `gli-flow telemetry`: Telemetry operations
- `gli-flow warehouse`: Telemetry warehouse management

# API Reference

This document lists the available API endpoints.

| Method | Path | Description |
| :--- | :--- | :--- |
| GET | `/runs` | List runs |
| GET | `/runs/{run_id}` | Run details |
| POST | `/failures/{failure_id}/resolution` | Record resolution |
| GET | `/analytics/summary` | Analytics summary |
| GET | `/analytics/common-failures` | Common failures |
| GET | `/atlas/trust-summary` | Atlas trust summary |
| POST | `/support-bundle/generate` | Generate support bundle |
| ... | ... | ... |
